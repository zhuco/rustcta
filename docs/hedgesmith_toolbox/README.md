# HedgeSmith 工具箱开发文档

## 项目名称与定位
- **正式名称**：HedgeSmith Toolbox（对冲匠工具箱）
- **一句话定位**：面向量化与主动交易员的多品类计算与辅助决策平台，覆盖网格、对冲、风险控制与资金曲线分析。

## 页面结构规划
- **首页仪表盘**：展示热门计算器快捷入口、网格策略快速模拟、最新功能更新与使用指南。
- **计算器一览页**：以分类与搜索聚合所有工具，支持收藏、常用排序与新手指引。
- **计算器详情页**：左侧参数输入（含预设模板与敏感度切换），右侧实时输出卡片与图表、风险提示、导出/复制功能。
- **数据面板**：整合现货/期货基差、波动率、资金费率等指标，可与计算器联动作为输入。
- **知识与案例库**：提供策略原理、操作清单、注意事项与FAQ，提升留存与SEO。
- **版本日志与反馈**：集中展示更新记录、用户反馈入口、数据来源与免责声明。

## 功能规划（首阶段目标 10 项）
1. **网格交易参数生成器**：根据价格区间、网格数量、资金规模自动计算每格委托价与投入资金比例。
2. **网格最大库存估算器**：在最极端行情下评估需要的最大持仓数量与资金占用，提示补仓/风控阈值。
3. **网格回撤分析器**：基于历史波动或自定义情景模拟最大回撤、回本周期、收益分布。
4. **现货-期货对冲比例计算器**：按基差、合约乘数与资金成本，输出最优套保比例与保证金需求。
5. **跨期/跨品种套利评估器**：支持输入多合约价格，计算价差、资金成本、目标收益与盈亏平衡点。
6. **仓位与杠杆管理面板**：整合账户权益、保证金率、强平线，提供杠杆调整与风控提示。
7. **风险收益比与止盈止损规划器**：根据入场价、止损距、目标收益自动计算RR、盈亏概率与盈亏金额。
8. **资金曲线与凯利公式模拟**：按胜率、赔率、期望收益模拟权益曲线、回撤与仓位建议。
9. **期权希腊值对冲助手**：输入期权链数据，输出 Delta/Gamma/Theta 等指标与对冲合约建议。
10. **事件驱动波动敏感度工具**：结合经济日历或自定义事件，估算事件前后波动区间与策略应对。

## 技术选型建议
- **前端**：Next.js + TypeScript + Tailwind CSS，初期所有计算在浏览器执行；后期可用 Zustand/Recoil 管理状态。
- **后端（可选）**：FastAPI + PostgreSQL / Redis，用于重型计算任务、记录用户模板、数据聚合。
- **共享计算核心**：Rust 编写计算逻辑，使用 `wasm-pack` 暴露到 Web；同一 crate 可复用到 Tauri 桌面端。
- **部署与运维**：前端托管在 Vercel，API 与任务工作器部署在 Fly.io；使用 GitHub Actions 运行测试与格式化。

## 数据与集成规划
- **实时数据**：对接交易所 REST/WebSocket（如 Binance、OKX、CME 数据供应商），以服务端缓存形式供前端调用。
- **离线批处理**：使用 `scripts/` 下 Python 工具拉取历史行情，存入对象存储用于回测或回撤分析。
- **配置管理**：所有 API Keys 置于 `.env`，使用 `serde_yaml` 验证新增配置，重要参数同步到 `config/` 目录。

## 开发阶段里程碑
1. **MVP（第 1-2 周）**：完成首页、计算器框架，落地网格三件套（参数、最大库存、回撤分析）。
2. **进阶对冲（第 3-4 周）**：上线现货-期货对冲、仓位风控、止盈止损规划，并接入基础行情数据。
3. **衍生品扩展（第 5-6 周）**：新增期权希腊值、套利评估，完善数据缓存与图表组件。
4. **分析与增长（第 7-8 周）**：实现资金曲线模拟、事件波动工具，上线收藏/模板功能与埋点。

## 设计与体验要点
- 参数面板采用分组折叠，提供默认模板与“复制上一组”功能。
- 结果区使用卡片+图表组合，突出关键指标，并附带风险提示。
- 对专业名词提供悬浮解释，全站支持中英文切换与暗色主题。
