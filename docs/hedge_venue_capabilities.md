# Hedge Venue Capabilities

Hedge recommendations must only include unified-account benefits when capabilities are explicitly verified.

Tracked venue fields include perpetual support, unified account support, spot-as-collateral support, cross/isolated/portfolio margin, internal transfer, collateral haircut knowledge, funding data, mark price, and index price.

Unknown collateral haircuts mean zero assumed unified-account benefit. One exchange's profit must never be assumed to support another exchange's margin.
