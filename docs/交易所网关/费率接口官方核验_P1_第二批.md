# 费率接口官方核验 P1 第二批

状态日期：2026-06-08

本批从剩余 `fee_official_check` 中收口 9 个 adapter。判断口径延续第一批：

- 官方文档明确有 maker/taker fee-rate readback，且当前项目没有稳定 `get_fees` runtime，写 `官方支持，项目未实现/未启用`。
- 当前 adapter 文档、endpoint mapping 或 runtime 明确把 fee readback 标成 unsupported、request-spec-only、未验证或零费率占位，写 `交易所不支持当前费率接口 runtime`。
- 只看到成交回报里的已成交手续费金额，不等于支持费率 readback；这类仍保留在剩余队列。

## 核验结论

| adapter | 产品线 | 官方/项目结论 | 项目当前状态 | 写法 |
| --- | --- | --- | --- | --- |
| `bybit` | Spot/Linear/Inverse/Option | Bybit V5 官方有 `GET /v5/account/fee-rate`，返回 maker/taker fee rate，支持 spot、linear、inverse、option 类别。 | 当前 `get_fees_impl` 仍返回 `unsupported_private("bybit.get_fees")`，endpoint mapping 未列标准 fee readback。 | 官方支持，项目未实现/未启用。 |
| `apex` | Perpetual | 当前 APEX profile 没有核到可映射成共享 `FeeRateSnapshot` 的稳定 private fee-tier endpoint。 | endpoint mapping 明确 `/unsupported/apex/fees`，runtime reason `apex.fees_private_read_not_promoted`。 | 交易所不支持当前费率接口 runtime。 |
| `bit2c` | Spot | 当前 Bit2C adapter 只保留 public REST 和 private request-spec 边界，私有 fees 明确不启用。 | adapter 文档写 private REST including fees unsupported；runtime reason `bit2c.fees_private_rest_disabled`。 | 交易所不支持当前费率接口 runtime。 |
| `bitbns` | Spot/Margin | 当前 Bitbns private fees 被关闭，未完成 KYC/region/account readback 验证。 | adapter 文档写 private balances、fees、open orders、query order、fills disabled；runtime reason `bitbns.fees_disabled`。 | 交易所不支持当前费率接口 runtime。 |
| `bitteam` | Spot/P2P boundary | 当前 Bit.Team private REST 仍是 request-spec-only，capabilities 默认把 fees 标为 unsupported。 | adapter 文档写 private REST、balances、fees、trading、cancel 等 unsupported；runtime reason `bitteam.fees_unverified`。 | 交易所不支持当前费率接口 runtime。 |
| `bitunix` | Spot/Perpetual | 当前 Bitunix public/private fee endpoint 未验证，项目只返回零费率占位，不能用于生产费率核算。 | adapter 文档写 fee endpoint was not verified；parser source `bitunix.*.fee_unsupported_default`。 | 交易所不支持当前费率接口 runtime。 |
| `coindcx` | Spot/Perpetual | 当前 CoinDCX adapter 未确认官方 fee endpoint，项目只返回零费率占位。 | adapter 文档写 fee endpoint not confirmed；parser source `coindcx.fee_endpoint_unconfirmed_zero_placeholder`。 | 交易所不支持当前费率接口 runtime。 |
| `tokocrypto` | Spot | 当前 Tokocrypto fee readback 仅保留 offline request-spec 边界，未提升为 runtime。 | runtime reason `tokocrypto.get_fees.offline_request_spec_only`。 | 交易所不支持当前费率接口 runtime。 |
| `zaif` | Spot | 当前 Zaif profile 明确不提供稳定 `get_fees` runtime。 | runtime reason `zaif.get_fees.unsupported`。 | 交易所不支持当前费率接口 runtime。 |
