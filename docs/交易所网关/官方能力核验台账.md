# 交易所官方能力核验台账

状态日期：2026-06-08

本台账记录已经查过官方资料的能力结论。它和 [交易所功能盘点矩阵](交易所功能盘点矩阵.md) 分工不同：

- 本台账回答：交易所官方到底支持什么。
- 功能盘点矩阵回答：当前项目已经声明/实现了什么。
- 没有查官方之前，不把“项目未声明”写成“交易所不支持”。
- 查实官方没有某产品线或能力后，单交易所文档必须写 `交易所不支持现货`、`交易所不支持合约` 或 `交易所不支持某接口`。
