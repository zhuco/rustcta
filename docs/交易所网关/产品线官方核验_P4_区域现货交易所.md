# 产品线官方核验 P4 区域现货交易所

状态日期：2026-06-08

本文件核验一批区域性现货交易所的合约/杠杆边界。目标是把“项目只声明 Spot”拆成三种结论：官方确实只有现货订单簿、官方有 margin/leveraged trading 但项目未实现、官方有标准合约需要单独补。

## 核验口径

- 标准合约口径指 futures、perpetual、option，以及能明确映射到共享合约仓位/资金费/强平/保证金模式的接口。
- Margin、leveraged trading、OTC、Convert、RFQ、Lending、Earn、支付/出入金不直接等同于标准 futures/perpetual/option。
- 官方 API 只列 spot market data、spot order、balance、withdraw/deposit 时，按当前官方 API 写 `交易所不支持合约`，并注明如官方上线 derivatives 需重核。

## 产品线矩阵

| adapter | 官方产品线结论 | 当前项目声明 | 结论和下一步 |
| --- | --- | --- | --- |
| `bitbank` | 官方 REST API 有 Spot order 路径，同时文档包含 Margin status/positions；未见标准 futures/perpetual/options API。 | `spot`。 | 写 `项目未实现 Margin`；标准 futures/perpetual/options 按当前 API 口径写 `交易所不支持合约`。 |
| `bithumb` | 官方 Open API 目录是 trading pairs、orderbook、ticker、orders、assets、deposit/withdraw、TWAP/batch orders；未见标准 futures/perpetual/options。 | `spot`。 | 写 `交易所不支持合约`。 |
| `bitopro` | 官方 Open API 覆盖全站交易对、order book、orders、wallet、REST/WS；未见标准 futures/perpetual/options。 | `spot`。 | 写 `交易所不支持合约`。 |
| `bitso` | 官方 Trading API 有 spot books/orders、OTC/RFQ/conversions，并列出 Margin Trading [WIP]；未见 futures/perpetual/options。 | `spot`。 | 写 `项目未实现 Margin [WIP]`；标准 futures/perpetual/options 写 `交易所不支持合约`。 |
| `btcmarkets` | 官方 API v3 覆盖 markets、orderbook、orders、account、fund transfer、batch orders；未见标准 futures/perpetual/options。 | `spot`。 | 写 `交易所不支持合约`。 |
| `coincheck` | 官方 Exchange API 是 Public/Private API，包含 order book、orders、balance、withdraw、WebSocket；另有 lending/account transfer；未见标准 futures/perpetual/options。 | `spot`。 | 写 `交易所不支持合约`。 |
| `coinone` | 官方 API v2/v2.1 目录覆盖 public market/orderbook/ticker、private balance/order/withdraw、public/private WS；未见标准 futures/perpetual/options。 | `spot`。 | 写 `交易所不支持合约`。 |
| `independentreserve` | 官方 features/API 说明有 spot order book/API，也有 leveraged trading。未见标准 futures/perpetual/options 统一 API。 | `spot`。 | 写 `项目未实现 Leveraged trading/Margin-like product`；标准 futures/perpetual/options 写 `交易所不支持合约`。 |
| `luno` | 官方 API 覆盖 accounts、market order book/tickers/trades/candles、orders、send/receive/transfers；页面无 futures/margin。 | `spot`。 | 写 `交易所不支持合约`。 |
| `upbit` | 官方 API reference 覆盖余额、交易对、订单、订单簿、ticker、withdraw/deposit、WS；未见标准 futures/perpetual/options。 | `spot`。 | 写 `交易所不支持合约`。 |

## 转入任务

| 类型 | adapter | 写法 |
| --- | --- | --- |
| 官方支持但项目未实现 | `bitbank` Margin、`bitso` Margin [WIP]、`independentreserve` leveraged trading | 写 `项目未实现`，但不要当作标准 futures/perpetual 直接接入。 |
| 明确交易所/API 不支持 | `bitbank`, `bitso`, `bithumb`, `bitopro`, `btcmarkets`, `coincheck`, `coinone`, `independentreserve`, `luno`, `upbit` 标准合约 | 单交易所文档写 `交易所不支持合约`。 |
