# 交易所适配器中文索引

状态日期：2026-06-08

这个索引用来解决“英文文件名不好识别”的问题。正式能力以每个 adapter 文档和 `endpoint_mapping.yaml` 为准；这里的中文说明只帮助快速定位。

## 快速口径

- 文件名保留英文 adapter id，是为了和代码目录、配置、fixture 一一对应。
- 产品线不存在时，在正式文档里写 `交易所不支持`，例如 `交易所不支持合约`。
- WebSocket 行情需要单独核对推流间隔、订单簿档位、订阅方式、sequence/checksum 和 snapshot 重建规则。
