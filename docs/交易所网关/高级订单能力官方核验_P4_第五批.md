# 高级订单能力官方核验 P4 第五批

状态日期：2026-06-08

本批收口 20 个 adapter。判定重点仍是 batch/amend/order-list 三类能力是否已经有项目证据，或是否能写成当前 profile 不支持。

## 核验结论

| adapter | 产品线 | 官方/项目结论 | 项目当前状态 | 写法 |
| --- | --- | --- | --- | --- |
| `bitget` | Spot | 当前 adapter 已有 amend runtime；batch place/cancel 走 composed sequential planning；order-list/OCO 未进入共享模型。 | `amend_order=运行`，top-level batch composed，但矩阵仍缺 batch 字段。 | 补矩阵识别 composed batch；order-list 写 unsupported 边界。 |
| `bithumb` | Spot | Bithumb Spot 普通 order/cancel 已映射；未确认 native amend、batch 或 order-list。 | endpoint mapping 明确 amend、batch place/cancel unsupported。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `bitmart` | Spot/Perpetual | BitMart futures batch cancel 已映射；batch place 计划单不等于普通 shared batch-place，amend/order-list 不进入共享模型。 | `batch_cancel_orders=运行`；batch place unsupported。 | 保留 batch cancel；batch place、amend/order-list 写 unsupported 边界。 |
| `bitmex` | Spot/Perpetual | BitMEX 已有 native bulk place/cancel 和 amend；标准 OCO/OTO order-list 不支持。 | `batch_place_orders=原生`、`batch_cancel_orders=原生`、`amend_order=原生`；queue 仅剩 order-list。 | 补/确认 order-list unsupported，保留 native batch/amend。 |
| `bitopro` | Spot | BitoPro private write/batch create/batch cancel 有官方路径和离线 request-spec；runtime live writes 仍 disabled。 | batch place/cancel spec_only native；amend unsupported。 | 补 batch live runtime 或保留 spec-only；amend/order-list 写 unsupported 边界。 |
| `bitrue` | Spot/Perpetual | Bitrue batch place/cancel 和 cancel-all 走 composed gateway flow；native batch、amend、OCO/order-list 不支持。 | batch composed sequential，矩阵只识别 batch cancel 运行。 | 补矩阵识别 composed batch place；amend/order-list 写 unsupported 边界。 |
| `bitso` | Spot | Bitso private REST 是 spec-only；native batch place unsupported，未建立 amend/order-list runtime。 | batch place explicitly unsupported；private write runtime 未启用。 | 写 `交易所不支持当前高级订单能力 runtime`；普通交易另按核心交易任务处理。 |
| `bitstamp` | Spot | Bitstamp adapter 已实现 amend；native order lists 和 batch place unsupported。 | `amend_order=运行`；order-list/batch unsupported。 | 保留 amend；batch/order-list 写 unsupported 边界。 |
| `bitteam` | Spot | BIT.TEAM private endpoints 是 request-spec-only；batch place/cancel 未验证，不进入 runtime。 | `place_order/cancel_order` 当前 unsupported；batch unsupported/unverified。 | 写 `交易所不支持当前高级订单能力 runtime`；普通交易另按核心交易任务处理。 |
| `bittrade` | Spot | BitTrade private order/cancel/query 有 request-spec，batch-cancel 有 request-spec；batch place、amend、cancel-all 不支持。 | `batch_cancel_orders=映射`；private live read/write 未提升。 | 补 batch cancel runtime 验收；batch place、amend/order-list 写 unsupported 边界。 |
| `bitunix` | Spot/Perpetual | Bitunix 已有 Spot/Perp native batch place/cancel；amend 和 OCO/order-list 不支持。 | `batch_place_orders=运行`，batch cancel 已有 native/spec 证据但矩阵 alias 不完整。 | 补矩阵识别 batch cancel；amend/order-list 写 unsupported 边界。 |
| `bitvavo` | Spot | Bitvavo 未声明 native batch endpoint；amend unsupported，order-list 不进入共享模型。 | adapter 文档写 batch not advertised、amend unsupported。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `blockchaincom` | Spot/Perp boundary | Blockchain.com private writes 多为 spec-only；batch place/cancel、amend 和 order-list 明确 unsupported。 | endpoint mapping batch unsupported；adapter 文档列出 batch/amend/order-list unsupported。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `blofin` | Perpetual | BloFin 已有 native batch place/cancel；amend 和 OCO/OTO order-list 不支持。 | `batch_place_orders=运行`、`batch_cancel_orders=运行`；order-list/amend unsupported。 | 保留 native batch；amend/order-list 写 unsupported 边界。 |
| `btcbox` | Spot | BTCBOX private writes 是 spec-only；无 native batch place/cancel，order-list runtime 不支持。 | batch place/cancel unsupported；private write offline request construction only。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `btcmarkets` | Spot | BTC Markets v3 batch orders 已有 native batch place/cancel request specs；amend/order-list unsupported。 | `batch_place_orders=运行`、`batch_cancel_orders=运行`；unsupported 列含 amend/order_list。 | 保留 native batch；amend/order-list 写 unsupported 边界。 |
| `btcturk` | Spot | BtcTurk private endpoints 是 spec_only；batch place/cancel unverified，不进入共享高级 runtime。 | top-level batch unsupported；private writes spec_only。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `bybit` | Spot/Linear/Inverse/Option | Bybit V5 batch create/cancel 官方能力未接入当前标准 trait；当前 adapter 只记录为 future work。 | batch create/cancel not wired；普通 order/cancel/cancel-all 运行。 | 写项目未实现/未启用 batch；不要写交易所不支持。 |
| `bybiteu` | Spot/Futures/Perpetual | Bybit EU profile 私有 REST/写接口未通过 broker scope 审计，所有 private trading/batch 维持 unsupported。 | place/cancel/cancel-all/batch 均 unsupported。 | 写 `交易所不支持当前高级订单能力 runtime`；区域凭证审计后再重核。 |
| `bydfi` | Perpetual | BYDFi 已有 amend、native batch place/cancel；order-list/OCO 不支持。 | `amend_order=运行`、`batch_place_orders=运行`、`batch_cancel_orders=运行`。 | 保留 amend/batch；order-list/OCO 写 unsupported 边界。 |
