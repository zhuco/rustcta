# 高级订单能力官方核验 P5 第六批

状态日期：2026-06-08

本批收口 30 个 adapter，覆盖剩余队列中 `cod3x` 到 `hollaex` 这一段。分类仍按“已有 batch/amend/order-list 证据或项目缺口”与“当前 profile 不支持高级订单 runtime”分开。

## 核验结论

| adapter | 产品线 | 官方/项目结论 | 项目当前状态 | 写法 |
| --- | --- | --- | --- | --- |
| `cod3x` | Perpetual routing | Cod3x 是 routing/terminal profile，未验证 Cod3x-native batch/amend/order-list API。 | place/cancel/batch/cancel-all 全部 unsupported_unverified。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `coinbase` | Advanced Trade/INTX | Coinbase adapter 已有 composed batch place、native batch cancel 和 trigger-bracket/OCO projection 线索。 | batch place/cancel 运行；矩阵仍缺 amend/order-list 字段。 | 补矩阵识别 batch/OCO 边界；amend/order-list 未无损映射的子能力单独写 unsupported。 |
| `coinbaseexchange` | Spot | Coinbase Exchange batch place/cancel 走 gateway-composed sequential calls；共享 amend/order-list 未映射。 | top-level batch composed；矩阵仍缺 batch 字段。 | 补矩阵识别 composed batch；amend/order-list 写 unsupported 边界。 |
| `coincheck` | Spot | Coincheck batch place/cancel 走 gateway-composed sequential calls；amend unsupported。 | top-level batch composed；矩阵仍缺 batch 字段。 | 补矩阵识别 composed batch；amend/order-list 写 unsupported 边界。 |
| `coinex` | Spot | CoinEx 已实现 amend；batch place/cancel 走 composed sequential。 | `amend_order=运行`；top-level batch composed 未被矩阵识别。 | 补矩阵识别 composed batch；order-list 写 unsupported 边界。 |
| `coinmate` | Spot | Coinmate 有 amend request-spec；native batch place/list cancel 不支持，cancel-all 仅作为 cancel-all。 | `amend_order=离线`；batch place/list cancel unsupported。 | 补 amend runtime 或保留离线；batch/order-list 写 unsupported 边界。 |
| `coinmetro` | Spot/Margin-like | Coinmetro 当前 adapter 不连接 batch/cancel-all、post-only/client order id 等高级交易能力。 | adapter 文档写 Batch/cancel-all unsupported。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `coinone` | Spot | Coinone Spot 普通交易运行；amend、order-list、batch place/cancel 明确 unsupported。 | unsupported 列含 amend_order、order_list、batch_place_orders、batch_cancel_orders。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `coinsph` | Spot | Coins.ph 现货普通交易运行；amend、order-list、batch place/cancel 明确 unsupported。 | unsupported 列含 amend/order_list/batch。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `coinspot` | Spot | CoinSpot side-specific order API 不能安全映射到 generic amend/query/batch。 | adapter 文档明确 cancel、amend、query-order、batch unsupported。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `coinstore` | Spot/Perpetual | Coinstore 已有 native batch place/cancel；amend 和 OCO/OTO order-list 不支持。 | `batch_place_orders=运行`、`batch_cancel_orders=运行`。 | 保留 native batch；amend/order-list 写 unsupported 边界。 |
| `cointr` | Spot/Perpetual | CoinTR 已有 Spot/Futures native batch place/cancel；amend 和 OCO/OTO order-list 不支持。 | `batch_place_orders=运行`、`batch_cancel_orders=运行`。 | 保留 native batch；amend/order-list 写 unsupported 边界。 |
| `coinw` | Spot/Perpetual | CoinW batch place/cancel 有 composed/native mixed 证据；amend 和 OCO/OTO order-list 不支持。 | batch place composed；perp batch cancel native；矩阵只识别 batch cancel。 | 补矩阵识别 batch place；amend/order-list 写 unsupported 边界。 |
| `cryptomus` | Spot | Cryptomus private writes 为 spec_only；batch place/cancel explicitly unsupported。 | top-level batch unsupported；private writes 未提升 runtime。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `deepcoin` | Spot/Perpetual | DeepCoin 已有 amend、native batch place/cancel；OCO/OTO order-list 不支持。 | `amend_order=运行`、`batch_place_orders=运行`、`batch_cancel_orders=运行`。 | 保留 amend/batch；order-list 写 unsupported 边界。 |
| `delta` | Futures/Option/Perpetual | Delta batch place/cancel 走 composed sequential；amend/order-list 未进入共享模型。 | top-level batch composed；矩阵仍缺 batch 字段。 | 补矩阵识别 composed batch；amend/order-list 写 unsupported 边界。 |
| `deribit` | Futures/Perpetual/Option | Deribit batch place/cancel 走 composed single-order endpoints；order-list/amend 未进入共享模型。 | top-level batch composed；普通 order/cancel 运行。 | 补矩阵识别 composed batch；amend/order-list 写 unsupported 边界。 |
| `derive` | Perpetual/Option | Derive private writes 仍是 spec_only/session-key 边界；native batch place/cancel 未验证。 | batch place/cancel unsupported_unverified；private writes 未启用。 | 写 `交易所不支持当前高级订单能力 runtime`，稳定 private write runtime 后再重核。 |
| `derive_chain_perps` | Perpetual profile | 该 profile 不承接 Derive Exchange API runtime；batch/amend/order-list 全部 unsupported_unverified。 | 所有 private write/batch endpoints 指向 unsupported profile。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `digifinex` | Spot/Perpetual | DigiFinex 已有 native batch place/cancel；amend 和 OCO/order-list 不支持。 | batch place/cancel 有运行/映射证据；amend/order-list explicit unsupported。 | 保留 native batch；amend/order-list 写 unsupported 边界。 |
| `dydx` | Perpetual | dYdX v4 链上交易支持 placing/replacing/canceling 线索，但当前 Node transaction signing 未实现。 | place/cancel/batch/amend/order-list 当前 Unsupported until Node write signing。 | 写项目未实现链上下单/改单/batch；不要写交易所不支持。 |
| `equation` | Perpetual | Equation 当前是 Arbitrum protocol audit profile，未验证 stable gateway API/indexer contract。 | batch/order writes unsupported_unverified。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `exmo` | Spot | EXMO Spot adapter 不暴露 native batch place/cancel、amend、OCO/OTO order-list。 | adapter 文档明确这些能力不暴露。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `fmfwio` | Spot | FMFW.io/HitBTC-family private trading runtime 仍 unsupported，未建立共享 batch/amend/order-list runtime。 | place/cancel 当前 unsupported；private endpoints request-spec-only。 | 写 `交易所不支持当前高级订单能力 runtime`；普通交易另按核心交易任务处理。 |
| `foxbit` | Spot | Foxbit private writes 为 spec_only；native batch place/cancel unsupported。 | top-level batch place/cancel unsupported；private writes 未提升 runtime。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `gateio` | Spot | Gate.io 已实现 amend；batch place/cancel 走 composed planning；order-list/OCO 未进入共享模型。 | `amend_order=运行`，top-level batch composed 未完全识别。 | 补矩阵识别 composed batch；order-list 写 unsupported 边界。 |
| `gemini` | Spot | Gemini Spot 未 advertised batch，当前 shared spot runtime 不暴露 composed batch/amend/order-list。 | adapter 文档写 Batch not advertised。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
| `hashkey_global` | Spot/Perpetual | HashKey Global batch place/cancel 走 gateway-composed flow；native batch、amend、OCO/OTO order-list 不支持。 | composed batch 未被矩阵完整识别。 | 补矩阵识别 composed batch；amend/order-list 写 unsupported 边界。 |
| `hitbtc` | Spot | HitBTC private trading runtime 仍 unsupported；未建立共享 batch/amend/order-list runtime。 | private endpoints request-spec-only；place/cancel 当前 unsupported。 | 写 `交易所不支持当前高级订单能力 runtime`；普通交易另按核心交易任务处理。 |
| `hollaex` | Spot/white-label profile | HollaEx demo/API profile private trading/readback 是 offline request-spec；runtime 仍 unsupported。 | private order/cancel/query/open/fills unsupported；batch/amend/order-list 未进入 runtime。 | 写 `交易所不支持当前高级订单能力 runtime`。 |
