# 费率接口官方核验 P0 第一批

状态日期：2026-06-08

本批先收口 `fee_official_check` 中证据明确的 50 个 adapter，并记录 1 个矩阵修正后已实现项。判断口径：

- 官方/当前 adapter 资料已经有 maker/taker、commission、交易费率表、账户费率、产品费率或链上参数费率来源，但当前项目未接 `get_fees` runtime、未补 mapping 证据，或只做默认/占位，写 `官方支持，项目未实现/未启用`。
- 当前 adapter/profile 不承接稳定共享费率接口，或费用属于下游 venue、链上资产费、经纪商费率表、停运 legacy venue 等不能映射成 gateway `FeeRateSnapshot` 的情况，写 `交易所不支持当前费率接口 runtime`。
- 只看到成交回报里的已成交手续费金额，不等于支持费率 readback；这类证据不足的 adapter 暂时留在剩余队列继续核验。
