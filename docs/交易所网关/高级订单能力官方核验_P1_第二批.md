# 高级订单能力官方核验 P1 第二批

状态日期：2026-06-08

本批继续收口 `advanced_trading_official_check` 中本地证据明确的 10 个 adapter。高级订单能力口径仍限定为：

- `amend_order`：改单、replace、modify。
- `place_order_list`：OCO/OTO/OTOCO 或订单列表语义。
- `batch_place_orders` / `batch_cancel_orders`：原生批量下单/批量撤单。

本批不把普通下单/撤单、cancel-all 或链上交易能力误记为高级订单支持。若官方/adapter 已确认普通交易可补，但批量、改单、OCO/OTO 没有稳定共享语义，则写 `交易所不支持当前高级订单能力 runtime`，普通交易继续按核心交易任务处理。

## 核验结论

| adapter | 产品线 | 官方/项目结论 | 项目当前状态 | 写法 |
| --- | --- | --- | --- | --- |
| `bitkan` | Spot/Perpetual | 当前未核到稳定 BitKan-native private trading API；高级订单全部是 unverified boundary。 | adapter 文档和 runtime 明确 `amend_order_unverified`、`order_list_unverified`、`batch_place_orders_unverified`、`batch_cancel_orders_unverified`。 | 交易所不支持当前高级订单能力 runtime。 |
| `zeta_markets` | Perpetual legacy | 原 Zeta Markets venue 已停止运营；legacy scan adapter 不应补 live private trading。 | place/cancel/amend/batch 均为 unsupported，venue shut down。 | 交易所不支持当前高级订单能力 runtime。 |
| `wavesexchange` | Spot | WX matcher 当前 profile 是 scan-only；matcher signed writes、批量、改单、订单列表不进入 shared gateway runtime。 | adapter 文档明确 batch place/cancel、amend、order lists unsupported。 | 交易所不支持当前高级订单能力 runtime。 |
| `d8x` | Perpetual | D8X 链上 perpetual 交易需要 EVM signer、ABI/calldata、gas/nonce/reorg 设计；当前没有 gateway-safe atomicity mapping。 | place/cancel/amend/order list、batch/cancel-all 均为 unsupported。 | 交易所不支持当前高级订单能力 runtime。 |
| `aftermath` | Perpetual | Aftermath Perpetuals 交易需要 Sui transaction 构造、签名和链上对账；当前 adapter 只做 public REST scan。 | batch trading、reduce-only、post-only、client order id 和 private runtime 均 disabled。 | 交易所不支持当前高级订单能力 runtime。 |
| `aark` | Perpetual | Aark/Orderly 普通交易是项目未实现核心交易任务；但当前 profile 未验证可映射到 shared amend/batch/order-list 语义。 | capabilities 明确 batch、cancel-all、amend、order-list flags 为 false；mapping 写 batch unsupported。 | 交易所不支持当前高级订单能力 runtime。 |
| `modetrade` | Perpetual | ModeTrade/Orderly 普通交易是项目未实现核心交易任务；但当前 profile 未验证可映射到 shared amend/batch/order-list 语义。 | batch place/cancel reasons 为 `unverified_orderly_semantics`，amend/order-list flags 为 false。 | 交易所不支持当前高级订单能力 runtime。 |
| `arkham` | Spot/Perpetual boundary | Arkham private REST 仍是 offline request-spec；未验证原生 batch place/cancel 或 OCO/OTO/order-list。 | runtime 明确 `arkham.amend_order_unsupported`、`order_list_unsupported`、batch unverified。 | 交易所不支持当前高级订单能力 runtime。 |
| `cex` | Spot | CEX.IO Spot Trading 支持普通 order/cancel 和 cancel-all，但未见 shared amend、OCO/OTO、generic batch place/cancel 语义。 | runtime 明确 `order_list_unsupported`、generic batch unsupported/unverified；普通 private writes 仍 request-spec-only。 | 交易所不支持当前高级订单能力 runtime。 |
| `p2b` | Spot | P2B private order/new 和 order/cancel 是 request-spec-only；未验证原生 batch、amend 或 OCO/OTO。 | runtime 明确 `p2b.amend_order_unsupported`、`order_list_unsupported`、batch unverified。 | 交易所不支持当前高级订单能力 runtime。 |
