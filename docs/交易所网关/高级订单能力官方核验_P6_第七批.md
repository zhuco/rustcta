# 高级订单能力官方核验 P6 第七批

状态日期：2026-06-08

本批收口剩余 38 个 adapter，覆盖 `htx` 到 `zebpay`。核验口径仍限定为批量下单、批量撤单、改单、OCO/OTO/order-list；普通下单/撤单、cancel-all、私有 WS 和费率不在本批重复展开。
