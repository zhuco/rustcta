# 产品线官方核验 P2 衍生品交易所

状态日期：2026-06-08

本文件核验一批衍生品、永续和链上订单簿交易所的产品线边界。结论只处理 Spot 是否应写成 `交易所不支持现货` 还是 `项目未实现 Spot`；其他核心接口、WebSocket 推流和私有接口仍以 [交易所功能盘点矩阵](交易所功能盘点矩阵.md) 和 [剩余官方核验队列](剩余官方核验队列.md) 为准。

## 核验口径

- 官方产品/API 已出现 spot、spot account、spot order book 或 spot trading 线索：写 `项目未实现 Spot`，不能写交易所不支持。
- 当前 adapter 是独立 perpetual/profile，且官方该 profile 只有永续/期权、没有可映射到共享 `Spot` 的订单生命周期：写 `交易所不支持现货`，并注明是 adapter/profile 口径。
- Swap、Earn、Stake、跨链兑换、钱包和充值提现不等同于共享 `Spot` 订单簿交易面。

## 产品线矩阵

| adapter | 官方产品线结论 | 当前项目声明 | 结论和下一步 |
| --- | --- | --- | --- |
| `apex` | ApeX Omni API 文档出现 Perps account 和 Spot account，且 account 响应包含 spot account/wallet 线索。 | `perpetual`；公共 perpetual REST 已实现，私有写入仍需 zkLink/L2 signer。 | 写 `项目未实现 Spot`；不要把 spot 写成交易所不支持。 |
| `apollox_dex` | APX/ApolloX V1/V2 主线是 perpetual；本地 adapter 文档也记录 ApolloX spot API family 属于另一个 API family。 | `perpetual`；V1 orderbook perp。 | 写 `项目未实现 Spot/当前 adapter 不接 spot API family`；不要按全品牌写交易所不支持。 |
| `aster` | Aster 官方 API 文档同时有 Spot API 和 Futures API。 | `perpetual`；只接 Futures V3。 | 写 `项目未实现 Spot`；后续可新增/扩展 Aster Spot mapping。 |
| `blofin` | BloFin 官方帮助中心和 Academy 明确有 spot markets/spot trading；当前 OpenAPI 文档以 SWAP/perpetual 为主。 | `perpetual`；当前 adapter 只接 USDT perpetual。 | 写 `项目未实现 Spot`，并单独核验 Spot API 下单/撤单是否公开稳定。 |
| `delta` | Delta 用户手册列出 Futures、Options、MOVE Options 和 Spot。 | `futures, option, perpetual`。 | 写 `项目未实现 Spot`；现有 Delta adapter 未声明 spot。 |
| `deribit` | Deribit API instrument kind 包含 `spot`，Support 也列出 Spot Instruments。 | `futures, option, perpetual`。 | 写 `项目未实现 Spot`；补 Deribit spot instrument/filter/orderbook/order lifecycle 前不要写交易所不支持。 |
| `derive` | Derive 官方 About 说明支持 options、perpetuals 和 spot trading。 | `option, perpetual`。 | 写 `项目未实现 Spot`；与 `derive_chain_perps` profile 的 `交易所不支持现货` 结论分开。 |
| `dydx` | dYdX 官方 2025-12-11 公告称 Solana spot trading 已上线，并和 perpetuals 同界面。 | `perpetual`；v4 Indexer perpetual markets。 | 写 `项目未实现 Spot`；需单独核验 Solana spot API/前端 routing 是否进入统一 gateway。 |
| `hyperliquid` | Hyperliquid 官方 Spot info endpoint 和 HIP-1 文档说明 onchain spot order books。 | `perpetual`；当前只接 `meta/l2Book/clearinghouseState` 等 perp 口径。 | 写 `项目未实现 Spot`；应补 `spotMeta`、spot asset ctx、spot order path/asset id 映射。 |
| `oxfun` | OX.FUN 官方文档主线是 Perpetual Swaps、automated strategies、vaults；未见共享 Spot 订单生命周期。 | `option,perpetual`；REST public market/orderbook 未确认，WS fixture/spec-only。 | 写 `交易所不支持现货`；期权/永续仍需独立补语义。 |
| `pacifica` | Pacifica 官方 Contract Specifications 是 perpetual contracts，API 目录围绕 perpetual markets、book、orders。 | `perpetual`。 | 写 `交易所不支持现货`；官方若以后上线 spot 再重核。 |
| `paradex` | Paradex 官方交易页列出 Perpetual Futures、Dated Options、Spot 和 Pre-markets。 | `option,perpetual`。 | 写 `项目未实现 Spot`；当前 adapter 只接 perp/options metadata。 |
| `woofipro` | WOOFi Pro 官方资料是 Orderly-powered decentralized perpetual futures exchange/API；WOOFi Swap 属于另一个产品，不是 WOOFi Pro Spot 订单簿。 | `perpetual`。 | 对 `woofipro` profile 写 `交易所不支持现货`；不要和 WOO X (`woo`) 或 WOOFi Swap 混用。 |

## 转入任务

| 类型 | adapter | 写法 |
| --- | --- | --- |
| 官方支持但项目未实现 | `apex`, `apollox_dex`, `aster`, `blofin`, `delta`, `deribit`, `derive`, `dydx`, `hyperliquid`, `paradex` | 单交易所文档写 `项目未实现 Spot`。 |
| 明确交易所/profile 不支持 | `oxfun`, `pacifica`, `woofipro` | 单交易所文档写 `交易所不支持现货`，其中 `woofipro` 按 WOOFi Pro profile 口径。 |
